# AI Chatbot Application

This project is an AI-powered chatbot developed for an organization.  
The frontend is built using **React** and **Tailwind CSS**, while the backend is developed with **Flask (Python)**.  
The chatbot is trained using **NLP techniques** with **NLTK**, and custom **intents** were created to handle organization-specific queries.
